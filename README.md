# Linux for DevOps

This repository contains my Linux learning journey for DevOps.

I am learning Linux step by step and documenting important concepts, commands, notes, and hands-on practice that are useful for DevOps, cloud infrastructure, automation, and server management.

## Why Linux for DevOps?

Linux is one of the core foundations of DevOps because most servers, cloud instances, containers, CI/CD tools, and automation workflows run on Linux-based systems.

## Topics Covered

- Linux basics
- Linux system architecture
- Kernel, shell, and bootloader
- Linux file system
- Basic Linux commands
- Users and permissions
- Process management
- Networking basics
- Bash scripting
- DevOps-related Linux practice
